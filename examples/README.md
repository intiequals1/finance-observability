# Examples

This directory contains public or synthetic example scenarios for finance observability.

## Planned examples

- `sec/` - SEC EDGAR and XBRL company facts
- `sap/` - SAP FI/CO and S/4HANA demo extracts using non-sensitive sample data
- `ifrs/` - IFRS-oriented validation and reporting examples
- `banking/` - Generic bank reconciliation examples using synthetic data
- `csv/` - CSV and Excel import patterns

## Data policy

Do not add confidential financial data, customer ledgers, production ERP exports, or private bank statements. Use public, synthetic, or anonymized data only.
