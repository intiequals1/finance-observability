# Roadmap

## v0.1 - Foundation

- Repository foundation
- Docker-based local development stack
- PostgreSQL schema for normalized finance observations
- Prometheus metrics endpoint
- Grafana provisioning
- Initial SEC/XBRL ingestion
- Basic data quality checks

## v0.2 - Finance Core

- Completeness checks
- Freshness checks
- Consistency checks
- Plausibility checks
- Source-to-target reconciliation patterns
- Materiality-aware thresholds
- Test data generators

## v0.3 - ERP and Reporting Examples

- SAP FI/CO demo extract format
- S/4HANA-oriented sample mappings
- Journal-entry quality checks
- Balance and movement validation examples
- CSV and Excel import examples

## v0.4 - Observability and Alerting

- OpenTelemetry traces for ingestion jobs
- Structured logging
- Grafana alert rules
- Data quality scorecards
- Pipeline latency monitoring

## v1.0 - Platform Release

- Stable module boundaries
- Documented connector SDK
- CI-tested example scenarios
- Production deployment guidance
- Security and governance documentation
