# Data Policy

Finance Observability must be safe to use and safe to share.

## Allowed data

- Public filings and open datasets
- Synthetic data
- Anonymized examples
- Demo extracts without personal or confidential information

## Not allowed

- Customer financial data
- Production ledgers
- Confidential ERP exports
- Private bank statements
- Credentials, tokens, keys, or secrets
- Personally identifiable information

## Repository rule

If a dataset cannot be published publicly without legal, contractual, or ethical concerns, it does not belong in this repository.
